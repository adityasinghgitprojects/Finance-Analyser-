# Finance-Analyser-
Personal Finance Dashboard

A Streamlit-based personal finance dashboard that allows users to upload their bank statements (CSV or Excel) and analyze their expenses visually. The app categorizes expenses automatically, provides summary metrics, and visualizes spending trends with interactive charts.

Features

Upload CSV or Excel bank statements

Automatic categorization of expenses (Food, Travel, Shopping, Bills, Entertainment, Others)

Summary metrics:

Total expenses

Average monthly expenses

Visualizations:

Category-wise expense pie chart

Monthly expense trend line chart

View detailed expense table

Technologies Used

Python 3.x – core programming language

Streamlit – for creating interactive web dashboards

Pandas – for data manipulation and analysis

Matplotlib – for creating charts and visualizations

Openpyxl – for reading Excel files
